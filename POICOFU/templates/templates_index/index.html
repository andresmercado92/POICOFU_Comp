Requisitos funcionales - Plataforma Online de interpretación de Código Fuente. (POICOFU).


Presentado por:
  • Andres Mercado
  • Julian Rueda
  1. Para poder usar los servicios de la plataforma el usuario tiene dos opciones: Una de las opciones es registrarse directamente en la plataforma. En este registro es obligatorio escribir los nombres, los apellidos, edad, ciudad de origen,país y lenguaje de preferencia. Una vez registrado el usuario puede modificar su información o darse de baja en el sistema. La otra opción es usar la cuenta de GMAIL. Para esto el sistema usa el servicio de autenticación de GMAIL quién es el que verifica la existencia del usuario y permite el ingreso a la plataforma.

  2. El sistema cuenta con un editor online donde el usuario escribe y compila los programas en el lenguaje de programación ofrecido por la herramienta.

  3. Este editor online permite guardar o persistir el código escrito en el disco duro del usuario y en la nube.

  4. Una vez compilado se verifica sintaxis, el léxico del lenguaje que reconoce la plataforma y se ejecuta el programa.

  5. El editor cuenta con una herramienta llamada “Ir” que permite ubicarse en cualquier parte del código.

  6. El Sistema visualiza estadísticas de todos los usuarios por lenguaje de preferencia y por país en su página principal.

  7. A cada usuario muestra el número de veces que ha compilado o interpretado los programas escritos sobre la plataforma.



Proyecto final para la asignatura Compiladores, Programa de Ingeniería de Sistemas Universidad del Magdalena, 2018-1.
